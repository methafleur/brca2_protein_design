# BRCA2 Protein Design with ProteinMPNN and LigandMPNN

Computational protein sequence design exercises on BRCA2 (breast cancer type 2 susceptibility protein)
## UniProt: P51587

## Tools used
- ProteinMPNN
- LigandMPNN (coming)

## Exercises completed
- Fixed backbone sequence design (Run 1)
- Partial sequence design with fixed DNA-binding residues (Run 2)

## Structure used
- 1MIU (3.1Å, BRCA2 DBD bound to DSS1, Yang et al. 2002)
